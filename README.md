## 🚀 How to Create a Free WordPress Website

### Method 1: WordPress.com (Easiest)

| Step | Description |
|------|-------------|
| 1 | Visit [WordPress.com](https://wordpress.com) |
| 2 | Click "Get Started" or "Start Site" |
| 3 | Register with email, Google, or Apple |
| 4 | Choose domain name (example: `namafaris.wordpress.com`) |
| 5 | Select **FREE** plan (scroll down) |
| 6 | Choose free theme |
| 7 | Verify email |
| 8 | Click "Launch Your Site" |

### Method 2: WordPress.org + Free Hosting (For Deeper Learning)

1. Find free hosting provider (AWS Free Tier, Oracle Cloud, Google Cloud)
2. Install WordPress on hosting (usually 1-click install)
3. Purchase domain (optional)
4. Customize freely (unlimited plugins, themes, and code)

## ⚠️ Free WordPress.com Limitations

| Limitation | Explanation |
|-----------|-------------|
| WordPress ads | "Website by WordPress" banner appears on site |
| Limited storage | Only 1GB available (enough for images, but limited) |
| No plugin installation | Premium plugins like WooCommerce not available |
| Limited customization | Fonts, colors, layouts cannot be freely modified |
| Subdomain only | URL becomes `namafaris.wordpress.com` |

## 🆚 WordPress.com vs WordPress.org Comparison

| Aspect | WordPress.com (Free) | WordPress.org + Hosting |
|--------|----------------------|------------------------|
| Analogy | Rental apartment | Empty land |
| Cost | Free | Paid hosting (or find free tier) |
| Domain | Subdomain `.wordpress.com` | Own custom domain |
| Plugins | Cannot install | Unlimited plugin installation |
| Ads | Yes | No |
| Ease | Very easy | Requires learning |
