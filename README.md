**AFSHOP** adalah website e-commerce sederhana yang menampilkan hypercar mewah eksklusif. Dibangun dengan **WordPress.com versi gratis** sebagai project sekolah.

### Konten Website:

| Halaman | Konten |
|---------|--------|
| Beranda | Informasi hypercar dan keunggulan investasi |
| Produk | Bugatti Chiron ($4 juta), Apollo EVO ($3 juta) |
| Kontak | Alamat, email, dan nomor telepon |

### Fitur:

- Desain responsif (tema bawaan WordPress.com)
- Tampilan produk dengan harga
- Form pemesanan (beli atau pesan)
- Informasi kontak lengkap

## 🚀 Cara Membuat Website WordPress Gratis

### Cara 1: WordPress.com (Paling Gampang)

| Langkah | Keterangan |
|---------|-------------|
| 1 | Buka [WordPress.com](https://wordpress.com) |
| 2 | Klik "Get Started" atau "Mulai Situs" |
| 3 | Daftar pakai email, Google, atau Apple |
| 4 | Pilih nama domain (contoh: `namafaris.wordpress.com`) |
| 5 | Pilih paket **FREE** (scroll ke bawah) |
| 6 | Pilih tema gratis |
| 7 | Verifikasi email |
| 8 | Klik "Launch Your Site" |

### Cara 2: WordPress.org + Hosting Gratis (Untuk Belajar Lebih Dalam)

1. Cari hosting gratis (AWS Free Tier, Oracle Cloud, Google Cloud)
2. Install WordPress di hosting (biasanya 1-klik install)
3. Beli domain murah (opsional)
4. Custom sesuka hati (bebas plugin, tema, dan kode)

---

## ⚠️ Keterbatasan WordPress.com Gratis

| Keterbatasan | Penjelasan |
|--------------|------------|
| Ada iklan WordPress | Banner "Website by WordPress" muncul di situs |
| Storage cuma 1GB | Cukup untuk gambar, tapi terbatas |
| Tidak bisa pasang plugin | Plugin premium seperti WooCommerce tidak bisa |
| Customisasi terbatas | Font, warna, layout tidak bisa diubah seenaknya |
| Subdomain wordpress.com | URL menjadi `namafaris.wordpress.com` |

## 📝 Catatan Penting

## 🆚 Perbandingan WordPress.com vs WordPress.org

| Aspek | WordPress.com (Gratis) | WordPress.org + Hosting |
|-------|------------------------|-------------------------|
| Analogi | Rumah kontrakan | Tanah kosong |
| Biaya | Gratis | Hosting berbayar (atau cari free tier) |
| Domain | Subdomain `.wordpress.com` | Bisa pakai domain sendiri |
| Plugin | Tidak bisa | Bebas pasang apapun |
| Iklan | Ada | Tidak ada |
| Kemudahan | Sangat mudah | Perlu belajar |
